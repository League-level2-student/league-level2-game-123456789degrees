Raining Tacos

by: Joshua Hsu

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game because I read a book about raining tacos and I thought it would be funny to make it like this.

2. What I Learned

I learned that even testers can sense errors that the developer can't. I got feedback that my game was too easy.

3. Architecture of My Game

There are the tacos and the rocks and the player, which all interact because if the player touches either the rock or the taco, the player would die or increase the score.

4. Technical Implementation

I basically got half of the code from League Invaders, because it is basically the same concept, but changed the projectile into the rock, which falls down at a faster speed than the taco. I am proud of the way the game keeps on getting harder as the game goes on.

5. Most Challenging Part

Making the images into ones that don't have a colored background. I had to ask the teacher about using GIMP and I forgot the steps several times.

6. Future Fixes and Updates

Maybe I would make the controls easier to control.
